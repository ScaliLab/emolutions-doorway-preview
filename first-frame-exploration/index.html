Fifteen first-appearance directions

Each remaining attempt holds to the same threshold line while changing the behaviour around it. Attempts 25-27 test perceptual demonstration: the frame makes noticing shift before explanation.

Attempt 01

Typographic stillness

The page gives the line almost nothing to lean on except timing, scale, and quiet.

Behaviour: blank arrival, delayed line, protected silence.

 Attempt 02

Receiving field

A subtle field begins first, as if the page has already made room for the visitor.

Behaviour: field first, line second, no bridge.

 Attempt 03

Delayed response

The line lands before anything else is allowed to answer it.

Behaviour: line first, pause, then upstream response.

 Attempt 10

Impact seam

Direct outcome blocks hit a hard seam; the sentence appears after impact.

Behaviour: saturated collision, seam holds, recognition follows.

 Attempt 13

Already there

The line is present immediately; the background only narrows attention around it.

Behaviour: instant line, peripheral attention, protected pause.

 Attempt 14

Plain spoken

The line arrives like a sentence being said, with no symbolic visual system to decode.

Behaviour: phrase cadence, bold colour, direct emphasis.

 Attempt 15

Attention draws in

The background gathers attention toward readable text instead of becoming a subject.

Behaviour: focus draws inward, line clears, pause line holds.

 Attempt 19

Outcomes first

The visible word outcomes gets first contact; the line names the habit around it.

Solution type: word priority and screen-space relation.

 Attempt 21

Recognition door

The visitor clicked because the line already felt true; the frame calmly confirms it.

Door type: recognition. Behaviour: calm confirmation.

 Attempt 22

Friction door

The visitor clicked because direct effort keeps meeting resistance before the outcome.

Door type: friction. Behaviour: resistance resolves into line.

 Attempt 23

Curiosity door

The visitor clicked because "upstream" was interesting but not yet understood.

Door type: curiosity. Behaviour: open loop, one step further.

 Attempt 24

Practical problem door

The visitor clicked because the pattern may apply to a real design or communication problem.

Door type: practical relevance. Behaviour: line as usable lens.

 Attempt 25

Misread first

The visitor first reads the situation one way, then sees that perception was shaped.

Demonstration: first perception shifts, then second beat resolves.

 Attempt 26

Attention redirected

A louder visible pull yields to an earlier source of attention.

Demonstration: attention moves from outcome pull to upstream source.

 Attempt 27

Outcome visible

An obvious outcome appears first; an earlier shaping point becomes visible after it.

Demonstration: outcome first, earlier cause revealed.
